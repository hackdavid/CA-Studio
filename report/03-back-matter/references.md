# References

## Placement

Include your reference list at the **end of your report but before the appendices**.

## Format

You **must** use the **IEEE referencing style**.

- See: [IEEE Citation Guidelines](https://ieee-dataport.org/sites/default/files/analysis/27/IEEE%20Citation%20Guidelines.pdf)

## Reference Management

It is highly recommended that you use reference management software such as **RefWorks**, which is provided by the university.

## Quantity

Your project should have **as many references as is required**. However, having **few references** indicates that no thorough investigation has occurred.

## Legitimacy

It is **your responsibility** to ensure that:

1. You have **actually read** all the material you reference.
2. The references provided in your report are **legitimate** and **NOT AI-generated**.

## EndNote Note

If you are using EndNote, your reference list will initially appear below the appendix. You can cut and paste it to the references page and it will stay there after that.

> **Do not delete** the hidden Section Break (Next Page) below this line. You can see hidden elements by using the **Show/Hide** option on the Home ribbon.
