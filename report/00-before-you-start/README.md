## Golden Rules

1. **Write in third person, observational perspective.** Never use "I", "we", "my", "our", etc. See [`writing-style.md`](writing-style.md) for the full banned-word list and approved phrasing.
2. **Follow the word-count limits** for each section. See [`structure-guide.md`](structure-guide.md).
3. **Use IEEE referencing style** for all citations.
4. **Ensure all referenced material has actually been read** and is legitimate (not AI-generated).
5. **Delete all template guidance text** (dark blue explanatory notes and red lines) before submission.
6. **Start each section on a new page.**
7. **Number all figures and tables** with captions.
8. **All text must be black** unless in exceptional circumstances.
