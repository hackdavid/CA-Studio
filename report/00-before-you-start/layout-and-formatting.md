# Report Layout & Formatting Rules

The following guidelines **must** be adhered to:

## Typography

| Element | Rule |
|---|---|
| Font | Arial or Calibri |
| Size | At least **12pt** for main body text |
| Colour | **All text must be black** unless in very exceptional circumstances |
| Line spacing | **1.5 line spacing** for main body text |
| Paragraph alignment | **Justified** |

## Page Layout

| Element | Rule |
|---|---|
| Margins | Minimum **2 cm** on each side |
| Page numbers | All pages must be numbered **consecutively** |
| Section starts | **Start each section on a new page** |

## Figures & Tables

| Element | Rule |
|---|---|
| Figures | Must have **captions** and be numbered (e.g., **Figure 1**) |
| Tables | Must have **captions** and be numbered (e.g., **Table 1**) |
| Colour | Figures may be black and white **or** colour |

## Captions & Automated Lists

- Use the **captions** feature (found in the References ribbon) for figures and tables.
- The **List of Figures** and **List of Tables** are automated from these captions.
- To update: right-click the list → **Update Field**.

## Template Cleanup

- Delete **all explanatory notes** in dark blue.
- Delete **all red guideline lines** before submission.
