# Report Structure & Word Counts

## Overall Structure

| Section | Guidance | Word Count |
|---|---|---|
| Declaration | See [`../01-front-matter/front-matter.md`](../01-front-matter/front-matter.md) | — |
| Acknowledgements | Thank supporters | — |
| Abstract | Short summary | Up to **500 words** |
| Table of Contents | Automated from Heading 1–3 | — |
| List of Figures | Automated from captions | — |
| List of Tables | Automated from captions | — |
| **Chapter 1 — Introduction** | Context, problem, objectives, methodology | Up to **1000 words** |
| **Chapter 2 — Literature & Technology Review** | Investigation of literature & technology options | Up to **1500 words** |
| **Chapter 3 — Implementation** | Design, sprints, difficult problems solved | Up to **3000 words** |
| **Chapter 4 — Evaluation & Results** | Strengths/weaknesses, metrics, related works | Up to **2000 words** |
| **Chapter 5 — Conclusion** | Summary, future work, reflection | Up to **1500 words** |
| **References** | IEEE format | — |
| **Appendices** | Proposal, management, artefact, screencast | — |

## Total Word Count Guide

There is a **10,000-word guide** for the project.

- All words in the **main body** count toward the total.
- Words in **figures and tables** are **excluded** from the word count.

## Page & Section Rules

- Start **each section on a new page**.
- All pages must be **numbered consecutively**.
- Additional chapters can be added using the **Heading 1** style.

## Automated Lists

- **Table of Contents** is automated based on Heading 1, 2, and 3 styles. Right-click → Update Field to refresh.
- **List of Figures** is automated based on captions using the "Figure" label.
- **List of Tables** is automated based on captions using the "Table" label.
